# AI & Automation Policy

`simpl-api` leverages AI tools to enhance productivity and maintain code quality.

## Usage

- **Assisted Coding:** AI assistants (like JetBrains AI, GitHub Copilot) are used for generating boilerplate code,
  writing tests, and suggesting refactorings.
- **Code Review:** AI-based tools are utilized to perform initial static analysis and detect potential security
  vulnerabilities or code smells before human review.
- **Documentation:** AI is used to draft documentation and maintain consistency across project files.

## Guidelines

- All AI-generated code must be reviewed by a human contributor for correctness, security, and adherence to project
  standards.
- Maintainers are responsible for the final integration and quality of any AI-assisted contributions.
